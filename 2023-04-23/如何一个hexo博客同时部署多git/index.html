    如何一个hexo博客同时部署多git

    <0x01> 如何实现

    其实可以通过本地多写几份 config 文件
    在部署时切换就可以

    <0x02> 开搞

    首先先新建几个 config 文件
    建议直接复制原来的_config.yml
    复制的文件名字随意
    我们只需要修改最下面的deploy部分就好

    比如我在 hexo 目录建了两个文件
    分别是_GitPageConfig.yml与_ServerConfig.yml

    然后新建一个脚本文件
    这里用 windows 配置，所以新建了个 powershell 脚本

    如果要部署到更多的git仓库，以此类推
    之后部署只需要执行deploy.ps1即可

    <0x03> 为什么不单文件解决

    (更新日期:23.07.07)
    经过我的实际测试，确实也可以直接单文件搞定

    这样写也可以，hexo会按从上到下的顺序依次部署

    但这样写会导致一个问题，就是网址的根域名会对不上

    如果仅靠单一文件的话，两个部署的根域名会是一样的
    这样可能会造成一些小问题

    靠powershell还可以顺便做部署前的一些操作，挺方便的
