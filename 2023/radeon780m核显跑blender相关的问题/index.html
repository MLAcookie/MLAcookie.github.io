    radeon780m核显跑blender相关的问题

    (@ 24-01-12)
    这篇博客目前写的有点简练，之后会再补充的

    <0x01> 先描述下问题

    最近入了台笔记本，7840HS的，CPU强核显也强
    （主要是没米了）
    然后毕竟是做游戏相关的，Blender总得装上吧
    然后发现，跑Cycles渲染器会报错
    HIP binary kernel for this graphics card compute capability (11.0) not found
    就会报这个错误

    解决办法

    从网站上下源码编译Blender就好
    Build Blender On Windows
    需要安装

    • VS2022的C++桌面开发工作负载
    • TortoiseSVN
    • Git
    • CMake
    • perl
    • python

    然后就按官方文章讲的，在cmd里面运行make update和make就好
    （powershell也不是不行，改成./make，但没试过）

    问题分析

    就是这么个问题
    radeon780m对应得代码是gfx1103
    但是blender正常下载的版本还没有带对应这个代码的驱动文件
    在最新的代码中，已经包含了生成代号gfx1103的驱动文件的代码了
    （其实就是CMakeList里面加了个gfx1103）
