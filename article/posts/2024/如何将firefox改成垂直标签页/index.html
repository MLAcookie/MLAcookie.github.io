如何将Firefox改成垂直标签页

Jan 02, 2024

我经常会开一堆网页，如果采用水平的标签页的话会经常找不到标签页
而火狐又不像别的浏览器，起码到目前为止都没有官方的垂直标签页的支持
经过一番探索，实际上是可以自己改出来的

<0x00> 效果展示


效果还是很不错的

<0x01> 安装Tree Style Tab插件

这是一个能在侧栏以树的形式管理标签页的插件，功能还挺多的
这个插件在火狐的插件商店里就有
别的类似的插件应该也有，但这里就用这个插件了

建议先熟悉下这个插件的操作再做下面的操作

<0x02> 打开about:config的一个选项

首先要打开about:config的关于自定义界面的选项
火狐会跳出提示，别动别的东西就好
搜索toolkit.legacyUserProfileCustomizations.stylesheets，把值改成true即可

<0x03> 打开标题栏

在火狐顶部右键，找到定制工具栏
（图片右上角）
然后打开左下角的标题栏

这样做是为了在隐藏顶栏后还能做拖动窗口之类的操作

<0x04> 找到用户的profile文件夹

这个也简单，Win+R输入%APPDATA%\Mozilla\Firefox\Profiles\
里面可能会有多个名字很乱的文件夹，问题不大，找到里面文件最多的文件夹
那个就是存用户设置的地方


（我这里是这样的）

<0x05> 创建修改界面的文件

在刚刚找到的文件夹里新建叫chrome的文件夹

然后在chrome文件夹里面新建一个userChrome.css的文件
打开编辑，输入下面的代码

/*侧边栏头部是否显示，会影响书签侧栏之类的，可以不写*/
#sidebar-header {
  display: none !important;
}
/*隐藏顶栏，也就是原来的标签栏*/
#TabsToolbar {
  visibility: collapse !important;
}

然后重启火狐理论上就生效了
